# Book Catalog Project Criteria

## Table Requirements Checklist

- [x] Create a table element for book information
- [x] Include a table head (`thead`) element with one row
  - [x] Row should contain 4 header cells (`th`)
    - [x] Title
    - [x] Author
    - [x] Genre
    - [x] Publication Year

- [x] Include a table body (`tbody`) element
  - [x] Minimum of 5 rows
  - [x] Each row should have 4 data cells (`td`) containing:
    - [x] Book Title
    - [x] Author Name
    - [x] Genre
    - [x] Publication Year

- [x] Include a table footer (`tfoot`) element
  - [x] One row with a single data cell
  - [x] Cell should span 4 columns
  - [x] Cell should display "Total Books: N"
    - [x] N should reflect actual number of books in table

## Additional Notes

- Ensure all data is properly aligned

- Verify column count consistency across all rows

- Double-check colspan attribute in footer
